Open Ocean VR
Virtual Reality Video Game

Capstone Project and Presentation for Senior Semester at University. Virtual Reality game designed for Oculus Rift using Unity Game Engine and C#. Player assumes role of deep-sea diver to interact and learn about the aquatic life around them. Created Game Event system to allow game designers to create quests using simple interface.

Royal Game of Ur
Mixed Reality Video Game

HCI Research Project under Mentor and Professor Krzysztof Pietroszek. Mixed-Reality interaction using a hand tracking Leap-motion sensor mounted on HMD. Programmed to allow player to use their bare hands to move pieces and play against competitor. AI opponent powered by a Min-Max Decision Tree and animated using Inverse Kinematics.
